# My GIT Cheat Sheet

## Git commands

`git log --pretty=oneline --abbrev-commit <path>` 

short hand: `git log --oneline <path>`

`git log  --graph <path>`

## Powershell Commands

|Powershell command| Unix equivalent|
|---|---|
| `get-help` | `man` |
| `remove-Item` | `rm` |
| `remove-Item -recurse -force ` | ` rm -rf ` |
